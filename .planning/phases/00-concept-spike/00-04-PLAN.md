<objective>
Prove pgvector can link entity mentions across items and languages: extract entities from the shared
corpus, embed them, and resolve one known entity across >=3 items / 2 languages into a single entity
without over-merging a distinct control entity. This resolves R3 and feeds ADR-006 (microservice
architecture + entity resolution). The production schema + Obsidian projection are Phase 8 (Deferred).

Purpose: no entities/entity_links schema or resolution exists yet; pgvector cosine linking is the
candidate mechanism Phase 8 will build on.
Output: schema SQL + NER + linking scripts + durable R3 verdict.
</objective>
