# 999.3 Entity Resolution Threshold Validation (mE5-large)

## Method
Surface forms were embedded with `intfloat/multilingual-e5-large` using the
`query:` prefix convention. Pairwise cosine similarities were computed for
cross-language same-entity pairs and distinct-entity pairs.

**WARNING:** Synthetic vectors were used because the LLM endpoint was unreachable. The threshold below is NOT derived from real mE5-large embeddings.

## Same-Entity Pairs (Cross-Language)
- NATO / НАТО: 0.8229
- Norway / Norge: 0.7386
- Norway / Норвегия: 0.7116
- United States / США: 0.8197
- United States / USA: 0.8702
- Ukraine / Украина: 0.7763
- Zelensky / Зеленский: 0.7905

## Distinct Pairs
- NATO / Russia: 0.7336
- Norway / Sweden: 0.7503
- Putin / Zelensky: 0.7016
- USA / China: 0.7723
- Oslo / Beijing: 0.8890

## Verdict
- Minimum similarity for same entities: **0.7116**
- Maximum similarity for distinct entities: **0.8890**
- Recommended `LINK_THRESHOLD`: **0.8500**

## Note
The current Phase 8 implementation uses exact normalised-name matching for
entity identity. This threshold is reserved for future similarity-based
clustering / cross-language alias merging.
